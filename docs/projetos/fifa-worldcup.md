# FIFA World Cup

Analise historica das Copas do Mundo (1930-2026) com dashboard interativo e modelos de previsao de desempenho de selecoes.

**Status:** Concluido | [GitHub](https://github.com/armandonettox/fifa-worldcup)

## O que faz

- Analise exploratoria dos dados historicos de todas as copas
- Identificacao de padroes de desempenho por selecao e periodo
- Modelo preditivo para estimar desempenho em edicoes futuras
- Dashboard interativo para navegacao dos dados e visualizacao de estatisticas

## Stack

- Python + pandas + numpy
- matplotlib + seaborn + plotly
- scikit-learn
- Streamlit
- Dados: Kaggle — FIFA World Cup Complete Dataset 1930-2026
